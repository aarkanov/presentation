                      Future of Mobile Testing at OTPP

                      • So where do we go from here??
                      • If we continue developing apps in the future they will get more complicated
                      • We will have lots of components that will have to be tested
                      • And automation will be essential to make sure that we can tests things quickly
                      • The new website is a 5 year project and we will have to make sure it meets its mobile first objective
                      • We will leverage all the tools that we have used so far to ensure succesfull testing on the new website
                      • Mobile and Web security issues are in the news on a daily basis
                      • We will have to make sure that our connected apps, and our mobile sites are secure
                      • We will also have to make sure that the member is informed regarding security issues
                      • More browser support
                      • We have users that are using pcs/macs/tablets/phones
                      • Android/iOS/Windows/Linux/Mac
                      • We need to make sure that our website are working, and are displaying properly
                      • on any device that a user logs in on

                      • In order to have access to all browsers we need to use tools such as Sauce labs
                      • Allows us to test across multiple browsers, and multiple operating systems
                      • They will also have a future open of testing on real devices over the cloud
                      • Performance testing will also be very important
                      • Most users these days have very high expecations when it comes to web and applications
                      • We will have to introduce new tools and make sure that our users get the optimal performance experience
                      • By 2017 most people will have at least 5 devices that are connected to the internet
                      • Most of you in the room most likely have at least 2
                      • We will have to make sure that the experience is the same accross all devices
